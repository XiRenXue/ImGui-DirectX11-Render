# Tips
- Dll项目中 我使用了ISO C++ 17 的规则
- 在创建窗口时 我将它设置为一个不可交互 透明 不显示在任务栏中的模式  并返回了窗口句柄 你可以通过这个窗口句柄来重新设置
- 同时返回了绘制线程的句柄 你可以通过这个句柄来做一些事情 或者手动关闭绘制
